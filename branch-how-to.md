Ветки в Git — это параллельные версии проекта, которые позволяют работать над разными задачами независимо. Вот основы:

Создать ветку:
git branch имя-ветки или сразу перейти: git checkout -b имя-ветки
Переключиться между ветками:
git checkout имя-ветки (или git switch имя-ветки)
Слить изменения из другой ветки:
В основной ветке выполни: git merge имя-ветки
Удалить ветку:
git branch -d имя-ветки
