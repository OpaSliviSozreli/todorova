# Инструкции по работе с SSH и GitHub

## 1. Как создать SSH ключ

1. Откройте терминал или командную строку.
2. Введите команду:
   ```bash
   ssh-keygen -t ed25519 -C "ваш_email@example.com"
  Если у вас нет поддержки ED25519, используйте:
  bash
  
  ssh-keygen -t rsa -b 4096 -C "ваш_email@example.com"
Следуйте подсказкам:

Укажите путь для сохранения ключа (обычно по умолчанию).
Введите пароль (опционально).
После завершения вы найдете ваш публичный ключ в файле:


~/.ssh/id_ed25519.pub
или


~/.ssh/id_rsa.pub
2. Как добавить ключ в аккаунт на GitHub
Откройте файл публичного ключа и скопируйте его содержимое:

bash

cat ~/.ssh/id_ed25519.pub
или

bash

cat ~/.ssh/id_rsa.pub
Перейдите на страницу GitHub и войдите в аккаунт.

Зайдите в Settings → SSH and GPG keys.

Нажмите кнопку New SSH key.

Вставьте скопированный ключ в поле Key.

Назовите ключ, например, "Мой ноутбук".

Нажмите Add SSH key.

3. Как склонировать репозиторий
Откройте терминал.
Перейдите в папку, куда хотите склонировать проект:
bash

cd путь/к/папке
Используйте команду:

git clone git@github.com:username/репозиторий.git
— замените username и репозиторий на свои данные.
